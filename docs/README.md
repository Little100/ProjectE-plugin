## 📦 核心功能

### EMC 系统
ProjectE 的核心是 EMC（Energy-Matter Covalence）系统。玩家可以将物品分解为 EMC，并使用 EMC 创造新的物品。

### 转换桌
使用转换桌可以将已学习的物品相互转换，只需消耗相应的 EMC。

### 能量收集器
自动从阳光中收集能量，为克莱因之星充能。

### 炼金术设备
包括能量凝聚器、炼金术箱子、暗物质熔炉等多种实用设备。

## 🎮 主要物品

- **贤者之石**: 核心工具，用于物品转换和合成
- **克莱因之星**: 存储 EMC 的容器
- **炼金术袋**: 便携式存储容器
- **暗物质工具**: 强大的工具套装
- **红物质工具**: 更强大的工具套装
- **宝石护甲**: 提供特殊能力的护甲
