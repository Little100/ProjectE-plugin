# 介绍

欢迎使用 ProjectE 插件！这是一个为 Minecraft Spigot/Paper 服务器设计的等价交换插件，灵感来源于著名的 ProjectE 模组。

## 什么是 ProjectE？

ProjectE 是一个基于等价交换概念的插件，它引入了一个核心的物质转换系统。玩家可以将物品分解为能量物质单位（EMC），并使用这些 EMC 来创造新的物品。

## 核心概念

### EMC（能量物质共价）

EMC 是 ProjectE 系统的核心货币。每个物品都有其对应的 EMC 值，这个值代表了该物品的"能量价值"。

- **学习物品**: 玩家需要先"学习"一个物品，才能在转换桌中创造它
- **转换物品**: 通过消耗 EMC，玩家可以将已学习的物品相互转换
- **存储 EMC**: 使用克莱因之星等工具存储大量的 EMC

### 转换桌

转换桌是 ProjectE 的核心设备之一。它允许玩家：

- 查看已学习的物品
- 使用 EMC 创造物品
- 学习新的物品

### 能量收集器

能量收集器可以从阳光中自动收集能量，为克莱因之星充能。有三个等级：

- **MK1**: 基础收集器
- **MK2**: 中级收集器
- **MK3**: 高级收集器

## 主要特性

### 🔮 完整的等价交换体验

- 与原版模组几乎相同的游戏体验
- 完整的 EMC 系统实现
- 支持自定义 EMC 值

### 🛠️ 丰富的工具和设备

- **贤者之石**: 核心工具，用于转换和合成
- **炼金术袋**: 16 种颜色的便携式存储
- **暗物质/红物质工具**: 强大的工具套装
- **能量凝聚器**: 自动将低价值物品转换为目标物品
- **炼金术箱子**: 大容量存储箱

### 💎 强大的装备系统

- **宝石护甲**: 提供特殊能力的护甲套装
- **护身符**: 各种提供被动效果的饰品
- **探矿杖**: 快速定位矿物

### 🎨 材质包支持

插件提供官方材质包和数据包，让你的服务器拥有与模组相同的视觉体验。

### ⚙️ 高度可配置

- 自定义 EMC 值
- 自定义配方
- 自定义设备参数
- 自定义权限系统

### 🌍 多语言支持

- 简体中文
- 英文
- 文言文（特色）
- 支持自定义语言文件

## 为什么选择 ProjectE 插件？

1. **完整的功能**: 实现了原版模组的绝大部分功能
2. **高性能**: 针对服务器环境优化，性能表现优秀
3. **易于使用**: 简单的安装和配置流程
4. **活跃维护**: 持续更新和修复问题
5. **社区支持**: 活跃的社区和详细的文档

## 系统要求

- **Minecraft 版本**: 1.16.5 - 1.21+
- **服务端**: Spigot 或 Paper（推荐 Paper）
- **Java 版本**: Java 17+（推荐 Java 21）
- **可选依赖**: 
  - Geyser（基岩版支持，实验性）
  - Folia（多线程服务端支持）
